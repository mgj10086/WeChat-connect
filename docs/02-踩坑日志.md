# 踩坑日志

> 记录运行过程中遇到的问题。

## 已知坑

### 1. daemon 找不到 claude 命令

**现象**：启动时报 `Failed to spawn claude`

**原因**：`claude` CLI 不在 PATH 中。

**解决**：
```powershell
# 确认位置
where claude

# 如果未安装：
npm install -g @anthropic-ai/claude-code
```

### 2. 扫码后 Bot 不回复

**原因**：
1. `context_token` 未正确回传
2. `AuthorizationType: ilink_bot_token` 请求头缺失

**解决**：检查 daemon 窗口的日志输出。

### 3. 回复模型不是 flash

**原因**：daemon 配置文件中的 model 未正确设置。

**解决**：编辑 `~/.wechat-claude-code/config.json`，确保包含：
```json
{ "model": "deepseek-v4-flash[1m]" }
```

### 4. ret:-2 限流

微信 API 限流，daemon 已内置指数退避，无需干预。

### 5. 关闭终端后 daemon 停止

daemon 运行在 PowerShell 窗口中，关闭窗口即停止。

**长期运行方案**：
- 使用 Windows Task Scheduler 创建任务，设置为"用户登录时启动"
- 或使用 `nssm` 将 daemon 注册为 Windows 服务
